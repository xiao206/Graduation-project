<p>一、毕业设计基本目标（根据自己使用的技术写）</p>
<p>综合运用所学的移动应用开发相关知识，面向宿舍/团体AA分摊场景，完成“宿舍AA分摊助手”移动端小程序的设计与实现。使用 Uni-app（Vue3 + TypeScript）完成界面与交互，使用 uni-ui 组件库提升开发效率，使用本地持久化存储（uni.setStorageSync/uni.getStorageSync）保存成员、分类、账单与设置等数据，实现账单记录、按参与成员分摊、生成结算明细与转账方案、分享结算文本等功能。依据需求文档完成核心功能模块，并完成系统功能测试与缺陷修复。</p>
<p>二、基本任务</p>
<p>对于完成本系统的设计与实现，具体任务要求如下：</p>
<ol type="1">
<li><p>完成产品需求分析，输出功能清单与关键业务流程说明；</p></li>
<li><p>完成数据来源说明：数据采用本地持久化存储，明确数据结构（成员、分类、账单、设置）与导入导出方式；</p></li>
<li><p>实现成员管理：新增/编辑/启用停用成员，支持默认参与成员配置；</p></li>
<li><p>实现分类管理：新增/编辑/排序/启用停用分类，用于账单归类统计；</p></li>
<li><p>实现账单管理：新增/编辑/删除账单，包含金额、付款人、参与人、发生时间、分类与备注；</p></li>
<li><p>实现结算模块：按月或自定义范围生成结算结果（已付、应摊、净额），生成最少转账次数的转账方案；</p></li>
<li><p>实现分享与数据：生成可复制的结算分享文本，支持数据导出/导入备份；</p></li>
<li><p>完成UI界面设计与移动端适配，优化留白与信息层级，提高可用性；</p></li>
<li><p>完成测试用例设计与功能测试，输出测试结论与改进点。</p></li>
</ol>
