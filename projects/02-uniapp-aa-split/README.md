# 宿舍AA分摊助手（UniApp）

一个本地优先的宿舍/同学 AA 分摊小程序：支持成员与分类管理、账单记录、自动分摊与结算、转账方案生成与分享，并提供数据导出/导入备份能力。

## 功能

- 成员管理：新增/编辑/启用停用成员，支持默认参与成员
- 分类管理：新增/编辑/排序/启用停用分类（颜色标签）
- 账单管理：新增/编辑/删除账单（金额、付款人、参与人、日期、分类、备注）
- 结算计算：本月/自定义范围，输出每人已付/应摊/净额
- 转账方案：生成“谁转给谁多少钱”，尽量减少转账次数
- 分享：一键复制结算文本，便于群聊同步
- 数据备份：导出/导入 JSON，支持跨设备迁移

## 技术栈

- UniApp（Vue 3）
- TypeScript
- 本机存储：`uni.setStorage` / `uni.getStorage`

## 开发

安装依赖：

```bash
pnpm install
```

运行到 H5：

```bash
pnpm dev:h5
```

运行到微信小程序：

```bash
pnpm dev:mp-weixin
```

类型检查：

```bash
pnpm type-check
```

构建（H5）：

```bash
pnpm build:h5
```
