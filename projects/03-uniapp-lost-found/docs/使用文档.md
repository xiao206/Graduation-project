## 1. 项目简介

“校园失物招领助手”是一个本地优先的 UniApp 小程序项目，用于校园场景下的寻物/招领信息发布与管理。项目不依赖后端服务，数据仅保存在本机 Storage 中，适合毕业设计演示与答辩展示。

## 2. 运行与构建

安装依赖：

```bash
pnpm install
```

运行到 H5：

```bash
pnpm dev:h5
```

运行到微信小程序：

```bash
pnpm dev:mp-weixin
```

类型检查：

```bash
pnpm type-check
```

构建（H5）：

```bash
pnpm build:h5
```

## 5. 核心使用流程

### 5.1 登录/注册

- 未登录时仍可浏览列表与统计
- 发布与“我的发布/状态管理/删除”等操作需要登录

### 5.2 发布信息

1. 进入“发布”页选择类型（寻物/招领）
2. 填写标题、分类、日期（必填）
3. 可选填写地点、联系方式、描述
4. 发布成功后进入详情页

### 5.3 列表检索

- “信息”页支持关键字搜索与类型筛选
- 条目显示状态：进行中/已认领/已结束

### 5.4 详情管理（仅发布者）

- 标记已认领：用于已找到失主/已取回
- 标记已结束：用于信息关闭
- 删除：删除该条信息

### 5.5 分类管理

- 新增分类后可在发布页选择
- 可启用/停用分类（停用后不影响历史数据展示）

## 6. 数据存储与清理

- 数据仅保存在本机 Storage 中
- H5 场景可通过浏览器“清除站点数据/清空缓存”来重置数据
- 小程序端可通过“清理缓存/卸载重装”来重置数据

## 7. 适配说明（H5）

- 在桌面宽屏下会以“手机壳”宽度居中展示，减少网页感
