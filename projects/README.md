## 命名建议

- `NN-项目短名`：`NN` 为两位数字递增（01、02、03...），便于排序
- 项目短名用小写与连字符：`01-web-demo`、`02-api-server`

## 每个项目建议包含

- `README.md`：目标、功能点、技术栈、启动方式、部署说明、演示链接
- `apps/`：前端/后端/管理端等多应用（可选）
- `packages/`：可复用组件/SDK（可选）
- `docs/`：项目内部文档（可选）
